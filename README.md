# Linux-Security-Hardening-Lab
## Project Overview
The project Linux-Security-Hardening-lab is a hands-on security assessment and hardening using linux commands where it goes through from network discovery to final security validation using ubuntu VM on macOs.The aim is to evaluate the system's security posture,vulnerability assessments,implement security controls and to monitor and audit by validating the improvements.
## Tools Used
Ubuntu Linux,Wireshark,Nmap,Lynis,UFW,Rkhunter
## Project phases 
1.Environment setup & Network Discovery
2.Traffic Analysis 
3.Security Auditing with Lynis tool 
4.Vulnerability Assessment
5.Authentication Log Review 
6.Firewall Hardening 
7.File Permission reviewing 
8.Service Hardening 
9.Malware and rootkit Detection 
10.System updates/patches 
11.Final Security audit/validation
